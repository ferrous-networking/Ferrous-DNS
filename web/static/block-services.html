Schedule Profile

A reusable collection of time slots with a timezone. Create one profile and assign it to multiple groups.

Time Slot

Defines which days and hours an action applies. Multiple slots can overlap — the most restrictive wins.

Group Assignment

Each group can have one active profile. The schedule evaluator runs every 60 s and updates overrides automatically.

🔴
Block All Blocks every DNS query for the group during this window — overrides blocklists completely.
🟢
Allow All Allows every domain — bypasses all blocklists. Useful for a scheduled "free access" window.
Typical example — "Kids" group
🔴 Mon–Fri 00:00–07:00 Block All
🟢 Mon–Fri 17:00–18:00 Allow All
🔴 Mon–Fri 21:00–23:59 Block All

Outside those windows the group's blocklists apply normally — gaming/social blocked, the rest works. The Allow All at 17h grants 1 h of unrestricted access even if gaming sites are in a blocklist.

Conflict rule: if a Block All and an Allow All slot overlap, Block All always wins — the most restrictive action takes priority.
